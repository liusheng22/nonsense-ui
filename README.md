# Nonsense UI (Vue 3)

这是一个专门制造“笑点”的 Vue 3 组件库。它的目标不是提升效率，而是通过反人类设计、视觉欺骗和物理折磨，制造极致的荒诞交互体验。

## 核心理念
- **反直觉**：打破所有现有的 UI/UX 准则。
- **幽默感**：让用户在挫败感中感受到一丝（荒诞的）快乐。
- **工程化**：虽然交互很无厘头，但代码和发布流程是认真的。

## 开发

```bash
pnpm i
pnpm dev:docs
```

## 构建与发布

```bash
pnpm build
# 发布到 npm
cd packages/ui
pnpm publish
```

## 文档部署
本项目使用 VitePress 驱动，适合部署到任何静态托管平台。
