# Nonsense UI (Vue 3)

**一个毫无用处、专门折磨用户的 Vue 3 荒诞组件实验室。**

在一个追求效率与生产力的时代，我们决定“反其道而行”。这里没有提效工具，只有解构现代交互准则的视觉欺骗、逻辑陷阱和物理折磨。

## 🧪 实验室准则
- **抛弃体验**：专注于制造令人愉悦的挫败感。
- **严谨乱来**：虽然交互很无厘头，但工程实现是极其认真的。
- **情绪价值**：唯一的目标是让你在受挫时笑出声。

## 开发

```bash
pnpm i
pnpm dev:docs
```

## 构建与发布

```bash
pnpm build
# 发布到 npm
cd packages/ui
pnpm publish
```

如果你有任何**反人类、反直觉、毫无逻辑**的组件点子，或者想为这个“荒诞实验室”贡献代码，我们非常欢迎！

- **提交 Issue**：分享你的脑洞（请确保它足够无厘头）。

- **讨论建议**：任何关于如何让交互变得更低效的想法。

让我们一起，在这个追求生产力的时代，制造一些毫无意义的快乐。
